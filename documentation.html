Project Overview

A comprehensive RESTful API for managing college operations including students, courses, enrollments, and payments.

🎯 Purpose

Streamline college management with automated workflows and REST APIs

🚀 Key Features

Complete CRUD operations, automated business rules, Swagger documentation

📊 Status

Production-ready with validation and error handling

🔗 Access

Port 8082 with Swagger UI at /swagger-ui

  • Swagger UI: http://localhost:8082/swagger-ui

Entity Details

1. Student Entity

Table: college.student

Java Class: com.college.cms.entity.Student

Field Type Constraints Description
id Long Primary Key, Auto-generated Unique identifier
name String NOT NULL Student name
address String - Student address
phoneNumber String NOT NULL Contact number (supports country codes)

2. Course Entity

Table: college.course

Java Class: com.college.cms.entity.Course

Field Type Constraints Description
id Long Primary Key, Auto-generated Unique identifier
title String NOT NULL Course name (e.g., "Mathematics")
code String NOT NULL Course code (e.g., "MATH-101")

3. Enrollment Entity

Table: college.enrollment

Java Class: com.college.cms.entity.Enrollment

Field Type Constraints Description
id Long Primary Key, Auto-generated Unique identifier
student Student Foreign Key, NOT NULL Reference to Student entity
course Course Foreign Key, NOT NULL Reference to Course entity
enrollmentDate LocalDate NOT NULL Date of enrollment
enrollmentStatus Enum NOT NULL PENDING, ACTIVE, COMPLETED, CANCELLED

Enrollment Status Workflow

  • PENDING Initial status after enrollment creation
  • ACTIVE Set automatically after successful payment
  • COMPLETED Course completed
  • CANCELLED Enrollment cancelled

4. Payment Entity

Table: college.payment

Java Class: com.college.cms.entity.Payment

Field Type Constraints Description
id Long Primary Key, Auto-generated Unique identifier
enrollment Enrollment Foreign Key, NOT NULL Reference to Enrollment
studentPhoneNumber String - Cached for search/compatibility
paymentDate LocalDate - Date of payment
paymentStatus Enum NOT NULL PENDING, SUCCESS, FAILED
amount Double NOT NULL Payment amount

Student API

Base Path: /students

Setup & Configuration

Prerequisites

  • Java 17 or higher
  • Maven 3.6+
  • PostgreSQL (any recent version)

Database Setup

Step 1: Create Database

Using psql:

psql -h localhost -p 5432 -U postgres -d postgres -c "CREATE DATABASE collegedb;"

Step 2: Configure Credentials

Default configuration in application.yml:

  • URL: jdbc:postgresql://localhost:5432/collegedb
  • Username: postgres
  • Password: postgres

Running the Application

mvn spring-boot:run

Application starts on http://localhost:8082
